TLV RPG — Sprite Sheet Layout Guide

Use this as your template when drawing in Piskel or PixilArt. Total sheet: 192 × 192 px, grid cell: 16 px.

How to use this guide:
1. Open Piskel / PixilArt and set canvas to 192 × 192 px
2. Enable the grid overlay at 16 px cell size
3. Draw each asset inside its labeled cell(s) — keep to the bounding box shown
4. Export as spritesheet.png and add it to the repo

Cell coordinates (col × row, 0-indexed, each unit = 16 px):
Player: col 0, row 0 — 16×16 px
NPC 1–4: cols 1–4, row 0 — 16×16 px each
Tiles (grass/sand/road/pavement/water): cols 0–4, row 1 — 16×16 px each
Wall/door tiles: cols 5–6, row 1 — 16×16 px each
Buildings (32×32 each, 2×2 cells): rows 2–5, cols 0–1 / 2–3 / 4–5 / 6–7
Palm tree + lamp post: col 8–9, rows 2–3 — 16×32 px each
Extras / decorations: row 6–7, any free cols
